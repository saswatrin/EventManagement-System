# Event Management System - Frontend Only

This is a standalone frontend application that works without a backend server.

## Quick Start

1. Navigate to the client folder:
   ```bash
   cd client
   ```

2. Install dependencies (if not already done):
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

4. Open http://localhost:3000 in your browser

## Features

✅ Login Screen - Enter any User ID and Password to login
✅ Vendor Signup - Register with name, email, password, and category
✅ Vendor Dashboard - Clean navigation interface
✅ Your Items - View list of items with edit/delete options
✅ Add New Item - Form to add items with category selection
✅ Transaction History - View transactions with chart toggle

## Usage

- Login with any credentials (no validation required)
- Sign up as a vendor with category selection (Catering, Florist, Decoration, Lighting)
- Navigate through dashboard options
- View sample items and transactions
- All data is mock data (not saved)
